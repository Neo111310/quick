# Zeilenumbuch im Mail Editor Ausschalten

Übersicht mit KI
To disable word wrap in the
Mozilla Thunderbird mail editor, open the Config Editor (Settings > General > Config. editor), search for mailnews.wraplength, and set its value to 0. For plain text, also set mail.wrap_long_lines to false. 
Steps to Disable Word Wrap:
-Open Config Editor: Go to Settings (or Tools/Options), scroll to the bottom of the General panel, and click Config. Editor.
-Modify mailnews.wraplength: Search for mailnews.wraplength and change its value to 0. This causes text to wrap to the window width instead of a fixed column.
-Disable Long Line Wrapping (Plain Text): Search for mail.wrap_long_lines and set it to false.
-lternative Option: Use the "Toggle Line Wrap" add-on to temporarily disable wrapping with a keyboard shortcut. 

Note: You may need to restart Thunderbird for these changes to take effect.
